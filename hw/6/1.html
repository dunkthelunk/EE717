Matrix Chain Multiplication

Due on 2015-11-07, 23:55 IST
You will be given N, the number of matrices, followed by N+1 entries that mark the dimensions of the N matrices for the matrix chain multiplication problem that we discussed in the class.


Write a C program that will print the optimal number of scalar multiplications that is needed to multiply the chain of matrices as well as the parenthesization of the matrices that result in the optimal choice. Use the dynamic programming technique that we discussed in the class.

INPUT:

A single integer N
N+1 integers

OUTPUT:

Single number indicating the optimal number of scalar multiplications
Fully parenthesized expression.

INPUT PROPERTIES:

1 <= N <= 100
Matrices are such that their dimensions do not exceed 1000.

Example:
